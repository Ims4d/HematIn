# HematIn

## Deskripsi
HematIn adalah aplikasi pengelolaan pengeluaran yang dirancang untuk bekerja secara offline. Aplikasi ini membantu pengguna mencatat, memantau, dan menganalisis pengeluaran harian mereka untuk mendukung manajemen keuangan pribadi yang lebih teratur.

## Fitur Utama
- Pencatatan pengeluaran harian dengan penyimpanan lokal menggunakan SQLite.
- Visualisasi data pengeluaran melalui grafik interaktif.
- Ringkasan pengeluaran dan rekomendasi anggaran.
- Sistem notifikasi pengingat untuk pencatatan rutin.
- Pengaturan preferensi aplikasi termasuk dukungan mode gelap dan terang.
- Penggunaan pola arsitektur MVVM (Model-View-ViewModel) untuk struktur kode yang bersih.

## Persyaratan Sistem
- Flutter SDK (>= 3.0.0)
- Dart SDK (>= 3.0.0 < 4.0.0)
- Perangkat Android atau Emulator

## Cara Setup
1. Unduh atau clone repositori ini ke direktori lokal Anda.
2. Buka terminal dan masuk ke folder proyek:
   ```bash
   cd HematIn
   ```
3. Jalankan perintah berikut untuk mengunduh semua dependensi yang diperlukan:
   ```bash
   flutter pub get
   ```
4. Hubungkan perangkat Android atau jalankan emulator.
5. Jalankan aplikasi menggunakan perintah:
   ```bash
   flutter run
   ```
